## Установка и запуск проекта локально

* Клонировать репозиторий
  
  `git clone git@github.com:SorokinaMarina/selsup.git`

* Перейти в папку "selsup"
  
  `cd selsup`

* Установить зависимости
  
  `npm i`

* Запустить 
  
  `npm run dev`
